Detailed Description

(package-private)
Manager of a reader associated to a "regular" plugin.

It contains all associated created card resources and manages concurrent access to the reader's card resources so that only one card resource can be used at a time.

(package-private)
Tries to lock the provided card resource if the reader is not busy.

If the provided card resource is not the current selected one, then tries to select it using the provided card resource profile extension.

Parameters
cardResourceThe card resource to lock.
extensionThe card resource profile extension to use in case if a new selection is needed.
Returns
True if the card resource is locked.
Exceptions
IllegalStateExceptionIf a new selection has been made and the current card does not match the provided profile extension or is not the same smart card than the provided one.

◆ matches()

std::shared_ptr< CardResource > keyple::core::service::resource::ReaderManagerAdapter::matches ( std::shared_ptr< CardResourceProfileExtension >  extension)

(package-private)
Gets a new or an existing card resource if the current inserted card matches with the provided card resource profile extension.

If the card matches, then updates the current selected card resource.

In any case, invoking this method unlocks the reader due to the use of the card selection manager by the extension during the match process.

Parameters
extensionThe card resource profile extension to use for matching.
Returns
Null if the inserted card does not match with the provided profile extension.
